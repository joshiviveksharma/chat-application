Java-based client-server chat application

## Overview
A real-time communication platform developed using Java that enables users to communicate over the internet. 
This project demonstrates the effective use of networking, multi-threading, and GUI design.

## Key Features
* Real-Time Messaging**: Facilitates instant communication between clients.
* Multi-threaded Architecture**: Supports multiple concurrent client connections.
* Secure Transmission**: Implements encryption to ensure data integrity and privacy.
* User-Friendly Interface**: Built with a GUI to provide an intuitive experience.


## Hardware & Software Requirements
* Processor**: Intel i5 and above 
* RAM**: 8GB and above
* Code Editor**: VS Code or Sublime Text 
* Version Control**: Git and GitHub 

## How to Run
1. Compile the server: `javac ChatServer.java`
2. Run the server: `java ChatServer`
3. Compile the client: `javac ChatClient.java`
4. Run the client: `java ChatClient`
